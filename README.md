# @metadev/essential-core

Metadev Essential/Core library.

Base clases for SDK for the meta-modeling framework **Essential**.

## Content

Base clases and interfaces for:

- ModelElement
- ModelBase
- Errors
- Parsing
- Linting
- Model Element Sequencer
